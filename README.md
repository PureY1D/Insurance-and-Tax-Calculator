# ITC 4.0.1 - 社保和个人所得税计算器

Insurance and Tax Calculator

## 项目简介

ITC 是一款专业的社保和个人所得税计算工具，支持企业职工社保、灵活就业社保、个人所得税等多种计算场景。

## 功能特性

### 三种计算模式

| 模式 | 说明 |
|------|------|
| **简易模式** | 输入即结果，实时计算，快速获取数据 |
| **学习模式** | 详细计算过程、政策说明、法律依据 |
| **配置模式** | 自定义社保比例、个税参数，自动保存 |

### 社保计算

- **企业职工五险一金**
  - 养老保险、医疗保险、失业保险、工伤保险
  - 住房公积金（可单独设置基数）
  - 实时显示到手工资、个人扣除

- **灵活就业社保**
  - 养老保险、医疗保险
  - 自定义缴费基数和比例

### 个人所得税计算

- **综合所得**
  - 工资薪金所得
  - 劳务报酬所得
  - 稿酬所得
  - 特许权使用费所得

- **经营所得**
  - 5级累进税率

- **比例税率所得**
  - 利息、股息、红利所得
  - 财产租赁所得
  - 财产转让所得
  - 偶然所得

- **扣除项目**
  - 基本减除费用（60,000元/年）
  - 个人社保（自动同步）
  - 专项附加扣除（子女教育、继续教育、住房贷款、住房租金、赡养老人、婴幼儿照护）

## 默认配置（宁波2024）

### 社保参数

| 参数 | 单位比例 | 个人比例 |
|------|----------|----------|
| 养老保险 | 14% | 8% |
| 医疗保险 | 7% | 2% |
| 失业保险 | 0.3% | 0.5% |
| 工伤保险 | 0.2% | - |
| 公积金 | 5%~12% | 5%~12% |

### 缴费基数

- 下限：4,986 元
- 上限：38,907 元

### 个税参数

- 基本减除：60,000 元/年
- 综合所得税率：7级累进（3%~45%）
- 经营所得税率：5级累进（5%~35%）
- 比例税率：20%

## 计算公式

### 社保计算

```
社保基数 = max(下限, min(上限, 工资))
公积金基数 = max(下限, min(上限, 公积金基数))

个人社保 = 养老(8%) + 医疗(2%) + 失业(0.5%) + 公积金
```

### 个税计算

```
综合所得 = 工资 + 劳务×80% + 稿酬×80%×70% + 特许权×80%

应纳税所得额 = 综合所得 - 60,000 - 个人社保 - 专项附加扣除

个税 = 应纳税所得额 × 税率 - 速算扣除数
```

### 到手工资

```
到手工资 = 税前工资 - 个人社保 - 个税
```
